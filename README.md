# Java-Shell

## System Requirements
Java (1.7 or later)

## Using the program

### Compiling
To compile the code, run the command bellow in the terminal
Note: Make sure you're running it from the root directory
```
$ javac driver/JShell.java
```

### Starting the Program

```
$ java driver/JShell
```

```
$ man [command]
```
